# **病人食雉肉喻**

##### 南朝萧齐天空三藏法师求那毗地 译

昔有一人病患委笃，良医占之云：

“须恒食一种雉肉可得愈病。”而此病者市得一雉，

食之已尽更不复食。医于后时见，便问之：

“汝病愈未？”病者答言：“医先教我恒食雉肉，

是故今者食一雉已尽更不敢食。”医复语言：

“若前雉已尽，

何不更食？汝今云何正食一雉望得愈病？”一切外道亦复如是，

闻佛菩萨无上良医说言，当解心识，

外道等执于常见，

便谓过去未来现在唯是一识无有迁谢，犹食一雉，

是故不能疗其愚惑烦恼之病。大智诸佛教诸外道除其常见，

一切诸法念念生灭，

何有一识常恒不变？如彼世医教更食雉而得病愈，佛亦如是，

教诸众生令得解，诸法坏故不常，续故不断，

即得铲除常见之病。
