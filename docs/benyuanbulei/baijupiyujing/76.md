# **田夫思王女喻**

##### 南朝萧齐天空三藏法师求那毗地 译

昔有田夫游行城邑，

见国王女颜貌端正世所希有，昼夜想念情不能已，

思与交通无由可遂，颜色瘀黄即成重病。诸所亲见，

便问其人：“何故如是？”答亲里言：“我昨见王女，

颜貌端正，思与交通不能得故，是以病耳。

我若不得，必死无疑。”诸亲语言：

“我当为汝作好方便，使汝得之，勿得愁也。”

后日见之便语之言：“我等为汝便为是得，

唯王女不欲。”田夫闻之欣然而笑，谓呼必得。

世间愚人亦复如是，不别时节春秋冬夏，

便于冬时掷种土中望得果实，

徒丧其功空无所获，芽茎枝叶一切都失。

世间愚人修习少福谓为具足，便谓菩提已可证得，

如彼田夫悕望王女。
