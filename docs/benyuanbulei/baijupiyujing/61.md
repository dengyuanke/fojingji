# **梵天弟子造物因喻**

##### 南朝萧齐天空三藏法师求那毗地 译

婆罗门众皆言：“大梵天王是世间父，

能造万物。”造万物主者有弟子，言我亦能造万物，

实是愚痴自谓有智。语梵天言：

“我欲造万物。”梵天王语言：“莫作此意，汝不能造。”

不用天语便欲造物。

梵天见其弟子所造之物，即语之言：“汝作头太大，作项极小，

作手太大，作臂极小，作脚极小，作踵极大，

作如似毗舍阇鬼。”以此义当知，

各各自业所造，非梵天能造，诸佛说法不著二边，

亦不著断亦不著常，如似八正道说法。

诸外道见是断见常事已，便生执著，

欺诳世间作法形像，所说实是非法。
