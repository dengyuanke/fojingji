# **尝庵婆罗果喻**

##### 南朝萧齐天空三藏法师求那毗地 译

昔有一长者，

遣人持钱至他园中买庵婆罗果而欲食之，而敕之言：

“好甜美者汝当买来。”即便持钱往买其果。果主言：

“我此树果悉皆美好无一恶者，汝尝一果足以知之。”

买果者言：“我今当一一尝之然后当取，

若但尝一何以可知？”寻即取果一一皆尝，

持来归家。长者见已恶而不食，便一切都弃。

世间之人亦复如是，闻持戒施得大富乐，

身常安隐无有诸患，不肯信之，便作是言：

“布施得福，我自得时然后可信。”

目睹现世贵贱贫穷皆是先业所获果报，

不知推一以求因果，方怀不信，须己自经，一旦命终，

财物丧失，如彼尝果一切都弃。
