# **奴守门喻**

##### 南朝萧齐天空三藏法师求那毗地 译

譬如有人将欲远行，敕其奴言：

“尔好守门并看驴索。”其主行后，时邻里家有作乐者，

此奴欲听不能自安，

寻以索系门置于驴上，负至戏处听其作乐。奴去之后，

舍中财物贼尽持去。大家行还，问其奴言：

“财宝所在？”奴便答言：“大家先付门驴及索，

自是以外非奴所知。”大家复言：

“留尔守门正为财物，财物既失用于门为？”

生死愚人为爱奴仆亦复如是，如来教诫常护根门，

莫著六尘，守无明驴看于爱索。

而诸比丘不奉佛教，贪求利养诈现清白静处而坐，

心意流驰贪著五欲，为色声香味之所惑乱，

无明覆心爱索缠缚，

正念觉意道品财宝悉皆散失。
