# **贫人烧粗褐衣喻**

##### 南朝萧齐天空三藏法师求那毗地 译

昔有一人贫穷困乏，与他客作，

得粗褐衣而被著之。有人见之而语之言：

“汝种姓端正贵人之子，云何著此粗弊衣褐？我今教汝，

当使汝得上妙衣服，

当随我语终不欺汝。”贫人欢喜敬从其言，

其人即便在前燃火，语贫人言：

“今可脱此粗褐衣着于火中，于此烧处，当使汝得上妙钦服。”

贫人即便脱着火中，既烧之后，

于此火处求觅钦服都无所得。世间之人亦复如是，

从过去身修诸善法得此人身，

应当保护进德修业，乃为外道邪恶妖女之所欺诳：

“汝今当信我语修诸苦行，投岩赴火舍是身已，

当生梵天长受快乐。”

便用其语即舍身命，身死之后堕于地狱备受诸苦，

既失人身空无所获，如彼贫人亦复如是。
